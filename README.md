# DDR AI System – Applied AI Builder Assignment

This project demonstrates an AI workflow that converts inspection and thermal reports into a structured Detailed Diagnostic Report (DDR).

## Workflow
PDF Input → Observation Extraction → Merge Logic → DDR Generation

## Files
- Sample Report.pdf – Inspection data
- Thermal Images.pdf – Thermal readings
- output/Main DDR(Detailed Diagnostic Report).pdf – Final generated DDR

## Note
API key required to run the LLM pipeline.
